# Evidence Query

Use this Skill for focused research questions about approved evidence in the
active release. Only use platform-provided published evidence search and source
citations. Do not infer clinical treatment advice, device instructions, or
unreviewed candidate knowledge.

Supported V1 question types include parameter summaries, safety and
contraindication checks, frequency-effect lookups, and direct evidence lookup.
Every substantive statement must be tied to a source chunk citation or state
that the published evidence is insufficient.
