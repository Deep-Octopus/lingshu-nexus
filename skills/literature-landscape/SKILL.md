# Literature Landscape

Use this Skill to summarize patterns in the active release rather than answer a
single narrow evidence lookup. It can aggregate published assertions and active
release source documents for research landscape questions.

Supported V1 question types include mechanism summaries, RCT design summaries,
timeline-style literature listing, research gaps, and broad landscape overviews.
If source documents do not contain publication dates or trial design metadata,
state that limitation instead of fabricating chronology or design details.
